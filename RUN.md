# Simple Run Commands

From project root (`/Users/adish/AI_Judge/`):

## Terminal 1 - Backend:
```bash
./start-backend.sh
```

## Terminal 2 - Frontend:
```bash
./start-frontend.sh
```

## Or manually:

### Terminal 1 - Backend:
```bash
cd backend
source venv/bin/activate
python main.py
```

### Terminal 2 - Frontend:
```bash
cd frontend
npm run dev
```

Then open: **http://localhost:5173**
